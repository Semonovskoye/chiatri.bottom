🔐 Công cụ mã hóa Question Bank

📌 Hướng dẫn sử dụng:

  1. Nhập mã khóa bí mật (lưu lại để dùng cho việc giải mã)
  2. Nhấn "🚀 Mã hóa TOPIC_ALL_TYPES ngay" để mã hóa dữ liệu đã nạp sẵn
  3. Hoặc dán code tùy chỉnh vào ô bên dưới rồi nhấn "Mã hóa Question Bank"
  4. Copy kết quả và thay thế vào file chính của bạn
  5. Bổ sung hàm getQuestions(type, difficulty) trong file chính để giải mã khi dùng
⚠️ Cảnh báo: Lưu lại mã khóa của bạn! Nếu mất mã khóa, bạn sẽ không thể giải mã được dữ liệu.
✅ Mã hóa thành công! Bạn có thể copy kết quả bên dưới.

📖 Cách tích hợp vào file quiz chính

Sau khi mã hóa, copy phần kết quả vào file HTML của bạn, sau đó sử dụng hàm getQuestions() để lấy câu hỏi:

Ví dụ: Lấy câu hỏi trắc nghiệm dễ của chủ đề "te-bao-co-ban"

const questions = getQuestions('te-bao-co-ban', 'mc');
// Lọc theo độ khó:
const easyQuestions = questions.filter(q => q.level === 'easy');

Các chủ đề (topic keys) trong TOPIC_ALL_TYPES:

  • 'te-bao-co-ban' — Tế bào cơ bản
  • 'thanh-phan-hoa-hoc' — Thành phần hóa học
  • 'quy-luat-phan-ly' — Quy luật phân ly
  • 'chu-trinh-calvin' — Chu trình Calvin
  • 'chu-ky-te-bao' — Chu kỳ tế bào

Các loại câu hỏi (type keys):

  • 'mc' — Multiple choice (Trắc nghiệm)
  • 'yn' — Yes/No (Đúng/Sai)
  • 'fill' — Fill in the blank (Điền chỗ trống)
  • 'match' — Matching (Ghép đôi)
  • 'essay' — Essay (Tự luận)
  • 'diagram' — Diagram (Sơ đồ)

Cấu trúc hàm getQuestions (đã được tích hợp trong kết quả mã hóa):

// Cách dùng:
const mcQuestions = getQuestions('te-bao-co-ban', 'mc');
const ynQuestions = getQuestions('chu-trinh-calvin', 'yn');
const fillQuestions = getQuestions('quy-luat-phan-ly', 'fill');

Ví dụ tích hợp vào hàm loadNextQuestionSet():

function loadNextQuestionSet() {
    let nextType = null;
    for (let type of state.selectedQuizTypes) {
        if (state.questionsPerType[type].questions.length < 5) {
            nextType = type;
            break;
        }
    }

    if (!nextType) {
        showCompletion();
        return;
    }

    state.currentQuizType = nextType;
    const typeData = state.questionsPerType[nextType];

    // SỬ DỤNG HÀM getQuestions ĐỂ GIẢI MÃ
    const questions = getQuestions(nextType, typeData.difficulty);

    if (!questions || questions.length === 0) {
        console.error('Không thể tải câu hỏi');
        alert('Lỗi: Không thể tải câu hỏi. Vui lòng kiểm tra lại.');
        return;
    }

    const randomIndex = Math.floor(Math.random() * questions.length);
    const question = questions[randomIndex];

    typeData.questions.push(question);

    displayQuestion(nextType, question, typeData.difficulty);
    updateStats();
}
